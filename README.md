## Calculator

A calculator built using HTML, CSS, and JavaScript. It has a smooth glass style design, and all the basic math functions you’d expect.

## Features

Modern glassmorphism UI

Basic operations (+, −, ×, ÷, %)

AC & DEL buttons

Smooth button animations

Works on all modern browsers

## How to Use

Open index.html in your browser and start using the calculator. Press = to see the result.
